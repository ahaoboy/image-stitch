# image-stitch

Online image stitching tool — join multiple images vertically or horizontally.

## Features

- Upload multiple images at once
- Drag split lines to crop each image
- Double-click a split line to toggle individual / batch mode
- Rotate, reorder, copy, delete images
- Apply filters (grayscale, blur, edge detection, etc.)
- Real-time WebGL preview
- Download as JPG or PNG
- Dark mode support

## Tech Stack

React 19 · TypeScript · MUI (Material UI) · Zustand · WebGL (regl) · i18next

## Getting Started

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
pnpm preview
```
